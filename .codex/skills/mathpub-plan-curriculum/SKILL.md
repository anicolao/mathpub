# Define a curriculum

Read [the curriculum contract](references/curriculum-contract.md).

Produce a curriculum map containing:

- stable unit and section identifiers;
- prerequisites and observable objectives;
- definitions, representations, and applications each section must teach;
- later opportunities to revisit each important skill;
- one full problem set and self-assessment per section; and
- one balanced practice and self-assessment per unit.

A topic counts as covered only when the student receives sufficient explanation, examples,
practice, and feedback to learn it from the book. Reject omitted prerequisites, abrupt difficulty
jumps, duplicated coverage, undefined vocabulary, and compressed survey sections.

Use the section, problem-set, self-assessment, and unit-practice skills to author the mapped output.
