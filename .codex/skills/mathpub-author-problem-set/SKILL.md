# Write a section problem set

Read [the problem-set standard](references/problem-set-standard.md).

Produce:

- clear directions addressed to the student;
- at least twenty distinct, self-contained mathematical questions;
- meaningful parts that progress from readiness and fluency to reasoning and application;
- workspace suited to the expected work;
- one short answer per numbered item; and
- one fully explained solution per numbered item.

Vary mathematical structure, representation, unknown position, difficulty, and context. Include
conceptual decisions and error analysis, but do not use follow-up commentary about the same four
problems to inflate the count.

Reject duplicate, cosmetic, degenerate, ambiguous, or out-of-scope questions. Require exact
one-to-one numbering across exercises, answers, and solutions.

Pair the set with $mathpub-author-self-assessment.
